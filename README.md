# THE BLOODLOVERS

A premium web application built with React, Vite, and Supabase.

## Features
- **AI Scanner**: Extract text and data from images using Tesseract.js.
- **Supabase Integration**: Real-time database and authentication.
- **Premium UI**: Modern design with Framer Motion animations and Tailwind CSS.
- **Dual Backend**: Python (for heavy processing) and Node.js servers.

## Getting Started

### Prerequisites
- Node.js
- Python (for backend)

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Setup backend (optional):
   ```bash
   cd backend
   pip install -r requirements.txt
   ```

### Running the App
- Start the frontend: `npm run dev`
- Start the backend: Run `run_backend.bat` or `start_app.bat`

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Framer Motion
- **Backend**: Python (Flask/FastAPI), Node.js
- **Database**: Supabase
